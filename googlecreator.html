Board Game Voice Creator (Google version)

Please download Python, and the script to convert your rules file into a Google voice compliant application.

Video Instructions, Part 1 Part 3 Download Python Script

 

Steps to follow:


1) Download Python application. This works on all operating systems, and will run a script that can convert your rules file into a Google Home voice application.

2) Once installed, download my python script

3) Copy the python script and your RULES.TXT file to the same directory

4) Open a command line, change to that directory and run "python convert_rules_to_google.py"

5) This will create a zip file for you to upload into Google's DialogFlow application, which will create a Google Home application. Please watch the video series linked above for more details
